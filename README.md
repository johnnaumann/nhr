# nhr

React + Vite app using **shadcn/ui**. The homepage renders the `login-03` screen.

## GitHub Pages

- **Live site**: `https://johnnaumann.github.io/nhr/`
- **Deploys**: on every push to `main` via `/.github/workflows/deploy-pages.yml` (builds `dist/` and publishes to Pages).

## Local development

```bash
npm install
npm run dev
```
